# Python Sets

Sets are unordered collections of **unique** elements.

## 1. Set Basics

```python
unique_nums = {1, 2, 3, 3, 4} # {1, 2, 3, 4}
```

## 2. Key Features

- **Unordered:** Items do not have a defined order.
- **Unique:** Automatically removes duplicate values.
- **Unindexed:** You cannot access items by index.

## 3. Set Operations

| Operation    | Operator |
| ------------ | -------- | --- |
| Union        | `        | `   |
| Intersection | `&`      |
| Difference   | `-`      |
