# Install Python (Windows)

Python is a versatile programming language. Follow these steps to install it on Windows.

## 1. Download Python

- Go to [python.org/downloads](https://www.python.org/downloads/)
- Click the download button for the latest version of Python.

## 2. Run the Installer

- Open the downloaded `.exe` file.
- **CRITICAL:** Check the box **"Add Python to PATH"** before clicking **Install Now**.
- Choose **"Customize installation"** if you want to change the install location, but most users can just use "Install Now".

## 3. Verify Installation

Open **Command Prompt** (cmd) and type:

```bash
python --version
```

You should see something like `Python 3.x.x`.
