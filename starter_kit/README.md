## üöÄ How to Use

1. **Install Dependencies**:

   ```bash
   uv sync
   ```

2. **Run an Example**:

   ```bash
   # Example: Run string manipulation
   uv run python 06_strings/basic_strings.py
   ```

3. **Run Tests**:
   ```bash
   uv run pytest
   ```
