# Install PyCharm

PyCharm is a powerful IDE (Integrated Development Environment) specifically designed for Python.

## 1. Download PyCharm

- Go to [jetbrains.com/pycharm/download](https://www.jetbrains.com/pycharm/download/)
- Download the **Community Edition** (it's free!).

## 2. Installation

- Run the installer.
- Follow the default prompts.
- When asked, you can check "Add launchers to PATH" and "Create Desktop Shortcut".

## 3. Setup Your First Project

- Open PyCharm.
- Click **New Project**.
- Select the Python interpreter (usually it detects the one you installed in the previous step).
- Click **Create**.
